# Фото-отчёт АЗС

Bitrix24-приложение для регулярного контроля порядка на АЗС через фото-отчёты.

MVP включает:
- расписание автоматических запросов отчёта;
- ручной запуск отчёта проверяющим;
- уведомления через бота `Порядок на АЗС`;
- пошаговую съёмку обязательных фото администратором АЗС;
- фоновую загрузку подтверждённых фото в Bitrix24 Диск;
- финальную отправку отчёта;
- рабочий стол проверяющего со статусами, фильтрами и быстрыми ссылками;
- ролевой доступ внутри одной встройки.

## Быстрый запуск разработки

```bash
make dev-node
```

После запуска приложение открывается через Bitrix24 iframe/placement. Настройка смарт-процессов, полей, стадий, ролей и расписания выполняется в разделе `Настройки` приложения.

## Важные ограничения

- Секреты, OAuth refresh token, Cloudpub token и client secret не должны попадать в Markdown-документацию и Git.
- Для стабильной работы scheduler нужен валидный per-user Bitrix24 OAuth-контекст администратора портала.
- Перед тестированием на портале приложение нужно переустановить после изменения scopes или bot/placement-настроек.
