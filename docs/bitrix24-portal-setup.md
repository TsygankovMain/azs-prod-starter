# Настройка Портала Bitrix24

Актуальная инструкция перенесена в spec-kit:

- [docs/spec-kit/03-bitrix24-setup.md](spec-kit/03-bitrix24-setup.md)

Этот файл оставлен как короткий alias, чтобы старые ссылки не ломались.

Коротко:
- подготовить СП `АЗС`, `Типы фото`, `Отчёты АЗС`;
- настроить поля и стадии;
- зарегистрировать бота `Порядок на АЗС`;
- выдать приложению scopes `crm`, `disk`, `im`, `imbot`, `placement`, `userfieldconfig`, `user`, `pull`;
- сопоставить СП, поля, стадии, роли и расписание в разделе `Настройки` приложения.
